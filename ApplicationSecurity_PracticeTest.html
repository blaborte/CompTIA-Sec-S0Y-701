This quiz covers Section 8: Application Security — SDLC phases & deployment environments (DEV/TEST/STG/PROD), DevOps vs SecOps, CI/CD pipeline stages, OWASP Top 10 Proactive Controls, secure database techniques (parameterized queries, tokenization, normalization), injection vulnerability types (SQLi, XSS, Command, LDAP, XML, HTML), application security techniques (SAST, DAST, fuzzing, code signing, HTTP headers), secure coding issues (race conditions, buffer overflow, memory management), session attacks, and additional attack types (CSRF, SSRF, RCE, Directory Traversal, File Inclusion, API attacks).

Tip: Know the difference between SAST (static — code not running) vs DAST (dynamic — code running). Parameterized queries are the primary defense against SQL injection. CSRF exploits the site's trust in the user; SSRF exploits the server to reach internal systems.
