Security+ Practice Exam A

A6.A company has hired a third-party to gather information about the company’s servers and data. This third-party will not have direct access to the company's internal network, but they can gather information from any other source. Which of the following would BEST describe this approach?
Correct Answer: B

Passive reconnaissance involves gathering information about a target without directly interacting with their systems or network, often using publicly available sources (OSINT).

A7.A company's email server has received an email from a third-party, but the origination server does not match the list of authorized devices. Which of the following would determine the disposition of this message?
Correct Answer: C

DMARC (Domain-based Message Authentication, Reporting, and Conformance) uses SPF and DKIM to determine the "disposition" (pass, quarantine, or reject) of an email.

A8.Which of these threat actors would be MOST likely to attack systems for direct financial gain?
Correct Answer: A

Organized crime groups are typically motivated by profit and direct financial gain through activities like ransomware and data theft.

A9.A security administrator has examined a server recently compromised by an attacker, and has determined the system was exploited due to a known operating system vulnerability. Which of the following would BEST describe this finding?
Correct Answer: A

Identifying the underlying reason an event occurred (in this case, an unpatched OS vulnerability) is the primary goal of root cause analysis.

A10.A city is building an ambulance service network for emergency medical dispatching. Which of the following should have the highest priority?
Correct Answer: C

For life-safety services like emergency medical dispatching, system availability is the most critical priority to ensure the service is always functional when needed.

A11.A system administrator receives a text alert when access rights are changed on a database containing private customer information. Which of the following would describe this alert?
Correct Answer: C

Automated alerts allow for immediate notification when specific security events occur, enabling faster response times.

A12.A security administrator is concerned about the potential for data exfiltration using external storage drives. Which of the following would be the BEST way to prevent this method of data exfiltration?
Correct Answer: A

Enforcing a policy at the OS level (like Group Policy in Windows) is the most effective way to prevent the system from even recognizing or mounting removable storage devices.

A13.A company creates a standard set of government reports each calendar quarter. Which of the following would describe this type of data?
Correct Answer: D

Government reports and compliance documents are considered regulated data because they must follow specific legal and professional standards.

A14.An insurance company has requirements for data handling: Save/archive access records, report access outside work hours, restrict access to the country, and create logs from a single database. Which THREE should be implemented?
Correct Answer: A, E, G

A addresses geographic restrictions; E consolidates logs into a single database for archiving/auditing; G enforces work-hour restrictions.

A15.A security engineer is viewing this record from the firewall logs:
UTC 04/05/2023 03:09:15809 AV Gateway Alert 136.127.92.171 80 -> 10.16.10.14 60818 Gateway Anti-Virus Alert: XPACK.A_7854 (Trojan) blocked.
Correct Answer: B

The log shows traffic from an external IP on port 80 (HTTP) being blocked by the Gateway AV due to a Trojan, indicating an attempted web download was stopped.

A16.A user connects to a third-party website and receives this message: "Your connection is not private. NET::ERR_CERT_INVALID". Which attack is the MOST likely reason?
Correct Answer: C

An invalid certificate error often indicates an on-path (Man-in-the-Middle) attack where the attacker is presenting a fraudulent certificate to intercept traffic.

A17.Which of the following would be the BEST way to provide a website login using existing credentials from a third-party site?
Correct Answer: A

Federation allows for the use of credentials across different networks or organizations, enabling users to log in to one site using the identity managed by another (e.g., "Log in with Google").

A18.A system administrator needs to know how often firewall hardware is expected to fail between repairs to specify a minimum uptime. Which of the following would BEST describe this information?
Correct Answer: A

Mean Time Between Failures (MTBF) is the predicted elapsed time between inherent failures of a mechanical or electronic system during normal system operation.

A19.An attacker calls a company’s help desk pretending to be a director to get a password reset quickly for a meeting. What kind of attack is this?
Correct Answer: A

Social engineering is the use of deception to manipulate individuals into divulging confidential or personal information, often using principles like authority and urgency.

A20.Two companies want to qualify their partnership with a broad formal agreement between both organizations. Which of the following would describe this agreement?
Correct Answer: C

A Memorandum of Agreement (MOA) is a document describing a cooperative relationship between two organizations wishing to work together on a project or goal.

A21.Which of the following would explain why a company would automatically add a digital signature to each outgoing email message?
Correct Answer: B

Digital signatures provide integrity by ensuring the message hasn't been altered in transit, and non-repudiation by verifying the sender's identity.

A22.A time clock appliance is configured to reset and reboot on error. A startup error now causes it to constantly reboot. Which issue is this?
Correct Answer: B

This cycle is a form of resource exhaustion/consumption where the system cannot complete its tasks, rendering it unavailable due to the constant reboot loop.

A23.Existing policies lack restrictions on password attempts and periodic changes. Which TWO would correct these?
Correct Answer: B, D

Password expiration forces periodic changes, and account lockout enforces a limit on failed password attempts.

A24.What kind of security control is associated with a login banner?
Correct Answer: B

A deterrent control discourages individuals from attempting a security violation by warning them of the consequences or policies.

A25.Unpatched servers are found and patching will take two weeks. What is the best quick response in the meantime?
Correct Answer: C

Isolating vulnerable servers in a protected or restricted network segment reduces the attack surface while testing for patches is underway.

A26.Steps for processing orders if the primary Internet connection fails are being documented. BEST describe these?
Correct Answer: B

Continuity of Operations (COOP) plans describe how an organization will continue to function during a disruption to normal operations.

A27.A company would like to examine the credentials of each individual entering the data center building. BEST facilitate this?
Correct Answer: A

An access control vestibule (mantrap) allows for the verification of credentials while the individual is held between two sets of doors.

A28.Employee info is encrypted but public details are plaintext. Which encryption strategy is this?
Correct Answer: B

Record-level encryption (or database encryption) allows specific fields or rows to be encrypted while others remain accessible in plaintext.

A29.A company would like to minimize database corruption if power is lost. BEST strategy?
Correct Answer: C

Journaling file systems or databases keep a log of changes that haven't yet been committed to the main part of the file system, allowing for recovery after a crash.

A30.Policy: Auto-lock, traceable location, and data separation. BEST way to establish these rules?
Correct Answer: D

Mobile Device Management (MDM) solutions allow administrators to enforce security policies like screen locks, tracking, and containerization (data separation).

A31.Monthly scan lists no vulnerabilities, but a significant unpatched one exists. Which of the following best describes this result?
Correct Answer: D

A false negative occurs when a security tool fails to report a vulnerability or threat that actually exists.

A32.An IT help desk is using automation to improve response time for security events. Which use case applies?
Correct Answer: A

Automation can be used to automatically escalate security events to the appropriate personnel based on predefined severity levels or triggers.

A33.An admin wants users to authenticate with corporate credentials for the wireless network. What should be configured?
Correct Answer: B

802.1X (WPA-Enterprise) allows users to authenticate against a central database (like Active Directory) using their unique corporate credentials rather than a shared key (PSK).

A34.A VPN performs a posture assessment during login. Which mitigation technique is this?
Correct Answer: D

Posture assessments enforce security configurations (like updated AV or OS patches) before allowing a device to connect to the network.

A35.A user assigns specific read-only access to three individuals for a file on their drive. Which access control model is this?
Correct Answer: A

In Discretionary Access Control (DAC), the owner of the data has the "discretion" to assign permissions to other users.

A36.A user receives a text message with a malicious login link. What BEST describes this attack?
Correct Answer: D

Smishing is a combination of "SMS" and "Phishing"—phishing attacks conducted specifically via text messages.

A37.A company is formalizing the design and deployment process for application programmers. Which policy applies?
Correct Answer: D

The Software Development Lifecycle (SDLC) defines the steps and policies for creating, testing, and deploying applications securely.

A38.An admin is running suspected malware in a sandbox. Which part of the incident response process is this?
Correct Answer: D

Analyzing malware in a sandbox helps determine the scope of the incident (Analysis) and ensures it does not spread further (Containment).

A39.Users are being directed to a different IP than the bank's actual web server. Which attack is most likely?
Correct Answer: D

DNS poisoning redirects traffic by corrupting DNS cache or records, leading users to malicious IP addresses instead of the intended site.

A40.Which consideration is MOST commonly associated with a hybrid cloud model?
Correct Answer: C

In a hybrid cloud, differing security controls between the on-premises and public cloud environments can lead to protection mismatches.

A41.An admin wants to verify that former employees cannot access systems. BEST way to provide this verification?
Correct Answer: C

Reviewing and validating offboarding processes ensures that the steps to disable accounts for departing employees are actually being followed and are effective.

A42.Which term describes how cautious an organization is regarding taking a specific risk?
Correct Answer: A

Risk appetite is the level of risk an organization is willing to accept in pursuit of its objectives.

A43.A tech patches a server and it fails with a critical error. What should the technician do NEXT?
Correct Answer: B

When a change fails during a maintenance window, the next immediate step is to follow the backout plan to restore the service to its previous stable state.

A44.An attacker crashes a server using crafted packets from many remote devices. What describes this?
Correct Answer: D

A Distributed Denial of Service (DDoS) attack uses many devices to overwhelm or crash a target system.

A45.An admin is building new servers and security systems to get financial systems back online after a breach. Which IR phase is this?
Correct Answer: C

Recovery involves restoring systems to normal operation and confirming they are functioning as expected after an incident.

A46.A wireless network was breached in less than 24 hours after new APs were installed. MOST likely reason?
Correct Answer: D

Misconfigurations (like leaving default passwords or keeping open access) are the most common cause of rapid breaches in new equipment.

A47.A firewall has a major vulnerability but is no longer for sale and won't be patched. Which BEST describes this?
Correct Answer: A

End-of-life systems are no longer supported or patched by the vendor, making them a significant security risk when new vulnerabilities are found.

A48.IT directors discuss a simulated disaster and the steps needed to resolve it. What BEST describes this exercise?
Correct Answer: D

A tabletop exercise is a discussion-based simulation where team members walk through their roles and responses to a hypothetical disaster.

A49.An admin needs to block users from malicious software websites. BEST way to control this access?
Correct Answer: C

DNS filtering blocks access to malicious domains at the name resolution level, preventing users from ever reaching the site.

A50.An admin re-images an infected system with a known-good version. Which IR step is this?
Correct Answer: B

Re-imaging a system to return it to a clean, working state is part of the Recovery phase of incident response.

A51.A company has placed a SCADA system on a segmented network with limited access. Which process describes this?
Correct Answer: D

Network segmentation is a common system hardening technique, especially for industrial control systems like SCADA, to reduce the attack surface and limit potential lateral movement.

A52.An administrator sees a log with hundreds of "Failed password for root" from the same IP. Which attack is this?
Dec 30 08:40:03 Failed password for root from 10.101.88.230 Dec 30 08:40:05 Failed password for root from 10.101.88.230 Dec 30 08:40:09 445 more authentication failures; rhost=10.101.88.230
Correct Answer: C

A brute force attack involves attempting many passwords against a single account (root) in rapid succession to gain access.

A53.A laptop was moved to a private VLAN and updates were automatically installed during login. What describes this?
Correct Answer: B

This is a form of Network Access Control (NAC) using configuration enforcement. The system checks for compliance (updates) and remediates the device before granting full access.

A54.Which of the following describes two-factor authentication (2FA)?
Correct Answer: D

Two-factor authentication requires two different categories: Something you know (password) and Something you have (smart card). Options A and C use only "Something you know."

A55.Concerns: No device management, many different types of devices, used for corporate and private use. Which model addresses this?
Correct Answer: D

Bring Your Own Device (BYOD) allows employees to use their personal mobile devices for business purposes, supporting a wide variety of hardware.

A56.An organization is installing a UPS for their new data center. BEST describe this control type?
Correct Answer: A

A UPS is a compensating control that provides power when the primary power source (the main electrical grid) fails.

A57.A company wants to track the progress of parts on an assembly line. BEST technology choice?
Correct Answer: B

Blockchain provides an immutable, distributed ledger that is ideal for supply chain tracking and verifying the history of parts or products.

A58.A website is replaced with a political message. MOST likely threat actor?
Correct Answer: D

Hacktivists are threat actors motivated by political or social causes, often using website defacement to spread their message.

A59.A download site shows an ISO and a SHA256 hash value. What is the use of this hash?
Correct Answer: A

Hashing is used to ensure integrity. By comparing the calculated hash of the downloaded file with the one provided by the site, you can verify the file is an exact copy.

A60.Policy requires login access only if physically within the building. BEST way to provide this?
Correct Answer: A

A physical USB security key requires the user to be physically present at the computer to plug in and tap the device to authenticate.

A61.A cloud database is available for anyone to query without authentication. Which vulnerability is this?
Correct Answer: B

Open permissions occur when access controls are not properly configured, allowing unauthorized users to read or modify data.

A62.Employees received an email with a link offering a cash bonus for completing training. BEST describe this?
Correct Answer: D

A phishing campaign uses deceptive emails to trick a large group of people into clicking a malicious link or providing sensitive info, often using "lures" like bonuses.

A63.Which risk management strategy includes the purchase and installation of an NGFW?
Correct Answer: B

Risk mitigation involves implementing security controls (like a Next-Generation Firewall) to reduce the likelihood or impact of a risk.

A64.All application requests must be validated at a policy enforcement point. BEST describe this?
Correct Answer: B

The Zero Trust model operates on the principle of "never trust, always verify," requiring every request to be authenticated and authorized at a PEP.

A65.What determines the assignment of data security in a public cloud infrastructure?
Correct Answer: C

The Shared Responsibility Matrix defines which security tasks are handled by the cloud provider and which are the responsibility of the customer.

A66.Decommissioning documents drive size, RAM, and adapter cards. Which process is this?
Correct Answer: D

Inventorying and documenting the specific components of a device during decommissioning is a form of enumeration or asset management.

A67.More info than expected in an API call allows arbitrary code execution. BEST describe this?
Correct Answer: A

A buffer overflow occurs when a program writes data beyond the boundaries of fixed-length blocks of memory, which can be exploited to execute code.

A68.Which of these keys should the organization place into escrow for backup?
Correct Answer: A

Key escrow involves storing a copy of a user's private key so that data can be decrypted by the organization if the original key is lost or for legal reasons.

A69.Host-based firewalls are enabled on all user devices. Which threat is addressed?
Correct Answer: D

Host-based firewalls help prevent on-path (Man-in-the-Middle) attacks by blocking unauthorized network traffic directly at the endpoint.

A70.Separate networks on the same physical switch without new hardware. BEST choice?
Correct Answer: D

Virtual Local Area Networks (VLANs) allow for logical segmentation of a network using the same physical hardware (switches and routers).

A71.Remote access for mobile field technicians to a cloud application. BEST option?
Correct Answer: D

Secure Access Service Edge (SASE) combines WAN capabilities with cloud-native security (like ZTNA and SWG) to provide secure remote access for mobile users.

A72.A company is implementing a quarterly security awareness campaign. Which of the following would MOST likely be part of this campaign?
Correct Answer: D

Awareness campaigns often involve reviewing and acknowledging the Acceptable Use Policy (AUP), which outlines the rules for using company assets.

A73.A recent patch reintroduced a vulnerability that was patched four months ago. What should be implemented to prevent this?
Correct Answer: D

Change management involves formal processes for testing and approving updates, which helps prevent "regression" where old vulnerabilities are accidentally reintroduced.

A74.Which of the following is the BEST way to add random data when generating a set of stored password hashes?
Correct Answer: A

Salting adds random data to a password before hashing, ensuring that identical passwords result in different hashes, which thwarts rainbow table attacks.

A75.Which cryptographic method is used to add trust to a digital certificate?
Correct Answer: D

A certificate authority (CA) uses its digital signature to sign a certificate, which allows others to trust that the identity on the certificate is valid.

A76.A company is using SCAP as part of their security monitoring. Which BEST describes this implementation?
Correct Answer: C

The Security Content Automation Protocol (SCAP) is a suite of specifications that standardize how security software communicates information about software flaws and security configurations.

A77.Who in an organization is responsible for managing the access rights to a large database of customer information?
Correct Answer: B

The Data Owner (typically a senior executive) is ultimately responsible for the data and determines who is allowed to access it.

A78.A CMS added a new classification type of “Private.” Which of the following would be the MOST likely reason for this?
Correct Answer: C

Adding specific classifications like "Private" helps organizations comply with privacy regulations (like GDPR or CCPA) by specifically marking Personal Identifiable Information (PII).

A79.A team wants to consolidate and protect private keys across all web servers. BEST way to securely store these?
Correct Answer: A

A Hardware Security Module (HSM) is a specialized physical device used for secure cryptoprocessing and centralized management of digital keys.

A80.Review the IPS log. Which TWO can be determined?
ALERT 2023-06-01 13:07:29 | Cross-Site Scripting in JSON Data 222.43.112.74:3332 -> 64.235.145.35:80 Detail: token="<script>" key="key7" value="<script>alert(2)</script>"
Correct Answer: B, C

The log shows an "alert(2)" script in the value field (B). The first IP address listed (222.43.112.74) is the source of the attack (C).

A81.Which of the following describes a monetary loss if one event occurs?
Correct Answer: B

Single Loss Expectancy (SLE) is the monetary value of any single loss. (ALE = SLE x ARO).

A82.A user types USER77' OR '1'='1 into a search field and all database records are displayed. BEST describe this?
Correct Answer: C

SQL Injection uses specifically formatted text to trick the database query into executing unauthorized commands (in this case, "OR '1'='1" evaluates as true for all rows).

A83.Issues began after opening a vendor spreadsheet. Symptoms: excessive popups, cursor moving independently. MOST likely cause?
Correct Answer: C

A Trojan horse is malware disguised as legitimate software (like the vendor spreadsheet) that contains hidden malicious functions like remote control access.

A84.Monthly charges are processed via saved credit cards. All info is encrypted and protected. What is the justification?
Correct Answer: C

Regulations like PCI-DSS require specific security controls for handling credit card information, which are verified through compliance reporting.

A85.Intermittent network communication occurs from workstations to one external IP at random times. MOST likely reason?
Correct Answer: B

Keyloggers or other malware often "beacon" data back to an external Command and Control (C2) server at varied intervals to avoid detection.

A86.An alert was received because credit card numbers were transmitted in an email attachment. MOST likely source?
Correct Answer: B

Data Loss Prevention (DLP) systems scan outgoing traffic for sensitive patterns (like credit card numbers) to block unauthorized data exfiltration.

A87.A VM in a screened subnet has a guest login and no password. MOST likely reason for this configuration?
Correct Answer: A

A honeypot is designed with intentional vulnerabilities to attract and monitor potential attackers to learn about their techniques.

A88.A security administrator is configuring a DNS server with an SPF record. What is the reason for this?
Correct Answer: B

SPF (Sender Policy Framework) is used to publish a list of all authorized email servers for a specific domain in its DNS records.

  • A is incorrect: Encryption is handled by the email server (TLS/SSL), not DNS records.
  • C is incorrect: DKIM is used for digital signatures.
  • D is incorrect: DMARC provides disposition instructions (reject, quarantine, etc.).
A89.A company wants to securely deploy applications without the overhead of a VM for each system. BEST way to deploy?
Correct Answer: A

Containerization allows isolated application instances to run on a single host OS kernel, providing efficiency and isolation without the resource overhead of multiple Virtual Machines.

A90.An admin wants to determine if a new server in a test environment can be exploited before production. BEST way to determine this?
Correct Answer: D

A Penetration Test involves active exploitation of vulnerabilities to see if they are actually viable. Because it can cause downtime, it is best performed in a test environment or during maintenance windows.

  • B is incorrect: Vulnerability scanners identify potential weaknesses but do not attempt to exploit them to confirm their impact.
